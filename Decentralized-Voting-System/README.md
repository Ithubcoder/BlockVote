# BlockVote ‚Äì Decentralized Blockchain Voting System

A secure, transparent, and tamper-proof voting system powered by **Ethereum Blockchain**, **Smart Contracts**, and **Web3.js**. This project eliminates the risks of manipulation present in traditional voting systems by ensuring **immutability**, **verifiability**, and **full transparency**.

---

## üöÄ Overview

BlockVote allows users to:

- Register as voters  
- Cast votes securely on the blockchain  
- View live results transparently  
- Prevent double voting automatically  
- Ensure every vote is immutable and publicly verifiable  

The system uses:

- **Smart Contracts (Solidity)**  
- **Truffle Suite** for development & deployment  
- **Ganache** as the local blockchain  
- **Web3.js** for frontend connectivity  
- **HTML/CCSS/JavaScript** for UI  
- **Optional Backend (Python API)** for data support
